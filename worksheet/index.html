상호작용 명령어

1. console.log

console.log("Hello, world!"); //Hello, world!
//undefined
  • 콘솔에 로그를 띄운다.

    • 2. alert

    alert("비밀번호가 일치하지 않습니다.");
  • 웹사이트에 경고창을 띄운다.

    • 3. prompt

    var answer = prompt("당신의 학번은?");
//(입력값)
  • 사용자의 정보를 입력받는 창을 띄우고, 입력된 값을 반환한다.

    • 변수 명령어

    1. var

    var classNumber = '30414';
//undefined
  • 가장 기본적인 변수 명령어
  • 같은 이름의 변수도 언제든 선언 가능하다.
  • 변수값도 언제든지 변경 가능하다.
  • 다른게 익숙하지 않다면 전부 이걸로 쓰는 게 처음에는 편하다.

    • 2. let

    let boolean = true;
//undefined
  • 익숙해지면 var보다 많이 쓰게 된다.
  • 같은 이름의 변수는 다시 선언하지 못한다.
  • 변수값은 변경 가능하다.
  • 블록 레벨 스코프이기 때문에 블록 내에서 선언되면 블록 내에서만 유효하다.

    • 3. const

    const canvas = document.getElementById('canvas');
//undefined
  • 같은 이름의 변수를 다시 선언하지 못하고, 변수값도 변경 불가능하다.
  • 또한 let과 같은 블록 레벨 스코프이다.
  • 제한된 사항이 많아 보통 주소 등을 저장할 때 쓰인다.

    • 데이터 타입

    1. 숫자형(Number)

    a. 정수

    100 //100
0 //0
-99 //-99

    b. 소수

    0.102 //0.102
0.1 + 0.2 //0.30000000000000004
  • 자바스크립트 특성상 오류가 나기 쉽다.
  • 중요하지 않은데에만 사용하자.

    • c. 무한

    Infinity //Infinity
Infinity + 1 //Infinity
Math.pow(10, 1000) //Infinity
Math.log(0) //-Infinity
1 / Infinity //0
1 / 0 //Infinity
  • 일정수를 초과하면 자바스크립트는 Infinity로 인식한다.
  • 수학적 무한과는 차이가 있다.
  • 코딩에서 실제로 잘 쓰지는 않는다.

    • d. NaN

    NaN //NaN
Infinity - Infinity //NaN
+"숫자 1" //NaN
NaN + 10 //NaN
'Ba' + NaN + 'a' //'BaNaNa'
  • Not a Number, 말 그대로 숫자가 아니다.(다만, 데이터 타입은 숫자형으로 분류한다.)
  • 오류가 떴을 때 자주 만나게 된다.
  • 자주 안보는게 제일 좋다.

    • 2. 문자열(String)

    '작은 따옴표' //'작은 따옴표'
"큰 따옴표" //"큰 따옴표"
`억음 부호` //`억음 부호`
  • 작은 따옴표(‘)나 큰 따옴표(“), 억음 부호(`)로 문자를 감싸서 사용한다.
  • 만약 이 부호들을 문자로서 사용하고 싶다면, 사용하고 싶은 부호 외의 부호로 감싸면 된다.
  • 그 외의 방법으로는 역슬래시(\)를 앞에 붙이면 된다.(\도 마찬가지로 \\로 쓰면 된다.)
  • 억음 부호는 줄바꿈(Enter)도 인식한다.

    • 3. 불리언(Boolean)

    true //true
        false //false
  • 참과 거짓을 나타내는 데이터 타입이다.

    • 4. 배열(Array)

    ['공수권', '김민수', '문재웅'] //['공수권', '김민수', '문재웅']
['공', '수', '권'][0] //'공'
['I', 'You', 'He/She'][1] //'You'
[0.1231, 1201, '5'][2] //'5'
[false, 2, true].length //3
  • 데이터의 정렬이다.
  • 각 데이터들은 순서에 따른 번호(인덱스)를 가진다.
  • 가장 앞에 있는 데이터가 0, 뒤로 갈 수록 1 씩 증가한다.
  • 예) 반에서의 번호에 따른 학생들의 정렬

    • 5. 객체(Object)

    let minsu = {
    name : "김민수",
    number : 5,
    male : true,
    friends : ["이진우", "염규원", "공수권"]
}
//undefined
minsu //{name: '김민수', number: 5, male: true, friends: Array(3)}
minsu.name //'김민수'
minsu.friends //(3) ['이진우', '염규원', '공수권']
  • 이름을 가진 데이터의 집합이다.
  • 객체 안의 데이터들은 각각 서로 다른 이름을 가진다.
  • 이름은 변수와 마찬가지로 숫자로 시작하면 안된다.
  • 예) 각 반의 학생

    • 6. 기타

    null
  • 값이 의도적으로 비어져 있는 경우
    • undefined
  • 값이 반환되지 않는 경우
